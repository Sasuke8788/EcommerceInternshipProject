# 🛍 E-Commerce Platform

A full-stack MERN-based E-Commerce platform with:
- 🔐 JWT Authentication
- 🛒 Cart & Stripe Checkout
- 📦 Order Tracking
- 🧑‍💼 Admin Product Management
- 🤖 AI-based Recommendations
- 🔍 Search & Filters

## 📚 Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Payment: Stripe
- Auth: JWT

## 🚀 Run Locally

### Frontend
```bash
cd client
